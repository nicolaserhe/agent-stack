# Skills — 地址簿

装 skill 时从这里找对应的大佬 repo。触发词 → repo 列表。

安装方式：按各 repo 自己 README 的说明装到 `<project>/.claude/skills/` 或 `~/.claude/skills/`。详见 [CLAUDE.md](../CLAUDE.md)。
